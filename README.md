<h1 align="center">Pomodoro</h1>

### Быстрый старт

```shell
make deps # установит все зависимости
make dev  # запустит приложение в режиме разработки
```

### Скрипты

```shell
dev           # запускает приложение в режиме разработки
start         # стартует приложение
build         # собирает приложение
build:analyze # запускает bundle-analyzer для сервера и клиента
type-check    # проверяет все TypeScript типы
format        # форматирует все файлы используя Prettier
format:check  # проверяет форматирование всеъ файлов согласно конфига Prettier
lint          # запускает ESLint
lint:fix      # проверяет все файлы согласно конфига ESLint
check:all     # запускает все проверки
```
